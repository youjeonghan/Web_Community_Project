## 사용 Stack
### Front
- HTML
- CSS
- Vanilla JS

### Back
- Python 3.8
- Flask 1.1.2
- Mysql 8.0

<br/>

## How to run?
 1. 가상환경 설치

```shell
virtualenv teamproject
```

2. requirements.txt 한번에 받기

```shell
pip install -r requirements.txt
```

3. DB이름은 flask_teamproject 생성해야한다.

4. 실행
```shell
python app.py
```
